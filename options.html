* If an external link host is included in the list but the current host page is not in the list the link is not replaced.
Like changing a twitter link "twitter.com/some_user" to an X link "x.com/some_user" in a current host "www.youtube.com".
To replace external links in such scenario, include a dummy host name belonging to the current page and give simple rules.
Ex:
Host Name = www.youtube.com
Patterns to Replace = //gi
Replacement = (empty)

If the current host name also has rules then this step is not necessary.
